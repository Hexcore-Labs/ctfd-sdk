# CTFd TypeScript SDK

A strongly-typed, Promise-based TypeScript client for the CTFd v3 API. This SDK provides a convenient wrapper around the standard CTFd REST API, allowing developers to interact with Users, Teams, Challenges, the Scoreboard, and more.

## Installation

```bash
# npm
npm install @hexcore-labs/ctfd-sdk

# yarn
yarn add @hexcore-labs/ctfd-sdk

# bun
bun add @hexcore-labs/ctfd-sdk
```
